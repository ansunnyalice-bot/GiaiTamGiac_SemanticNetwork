# 📐 PHẦN MỀM GIẢI TAM GIÁC - MẠNG NGỮ NGHĨA

Dự án môn học Trí Tuệ Nhân Tạo - Xây dựng chương trình giải toán tam giác dựa trên mô hình mạng ngữ nghĩa.

## ✨ Chức năng chính
- **Tính toán:** Nửa chu vi, Diện tích, Bán kính nội tiếp (r), Bán kính ngoại tiếp (R).
- **Ràng buộc:** Kiểm tra điều kiện ba cạnh có tạo thành tam giác hay không.
- **Mạng ngữ nghĩa:** Tự động cập nhật ma trận quan hệ (giá trị -1) khi tính toán.

## 🛠 Công nghệ sử dụng
- Ngôn ngữ: **C#**
- Giao diện: **WPF (.NET)**
- Kiến trúc: Semantic Network Logic
